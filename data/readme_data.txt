Data Acquisition Instructions:

To run this project locally:

Download the "CMAPSS Jet Engine Simulated Data" from Kaggle: https://www.kaggle.com/datasets/palbha/cmapss-jet-engine-simulated-data or from NASA: https://data.nasa.gov/docs/legacy/CMAPSSData.zip.

Extract the zip file.

Place the following files directly into this /data folder:
train_FD001.txt, test_FD001.txt, RUL_FD001.txt
train_FD002.txt, test_FD002.txt, RUL_FD002.txt
test_FD004.txt, RUL_FD004.txt
